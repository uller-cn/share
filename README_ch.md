# 关于

**Share** 是一个局域网文件共享小部件，支持PC自动发现、移动计算机相互传输和无限文件大小传输。

---

# Features

- [x] 局域网PC自动发现
- [x] 共享文件自动发现
- [x] 文件互传签名验证
- [x] 手机扫码互传
- [x] 局域网共享文件搜索
- [ ] 支持mac系统
